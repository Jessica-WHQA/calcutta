DOWNLOAD CALCUTTA FOR ANDROID

To download Calcutta for Android, click the download button.

Once the file has downloaded open your file manager, and tap on the file to open it.

If prompted, you will need to allow installation from unknown sources:

  • Go to Settings > Security and Privacy > More security settings > Install unknown apps.
  • Select your File Manager and enable "Allow from this source".
  • Select your default browser and enable "Allow from this source"

Return to your file manager and tap the file to open it.

When prompted tap the Install button

Once installed, open Calcutta and sign in via your Grint account & start making wagers on your golf games!
